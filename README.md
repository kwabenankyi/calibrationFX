# calibrationFX

## Instructions

1. Create virtual environment, activate it, and run `pip install -r requirements.txt` to install all dependencies.

2. Run `main.py` for the full optimisation loop (might take a while!), or to view results for any **valid** parameter combination with the market data and the quadratic rough Heston model, use the notebooks `qr_heston_sim.ipynb`, `qr_heston_results.ipynb`, ensuring you have set the correct variables in `fx_init_const.py`.
